# Sistema de Gestión Empresarial

Sistema CRUD completo desarrollado con arquitectura de 4 capas y programación orientada a objetos.

## Arquitectura

- **Presentación**: Interfaces de usuario y menús
- **Aplicación**: Lógica de negocio y reglas
- **Dominio**: Modelos de datos (DTOs)
- **Persistencia**: Acceso a datos (DAOs)

## Funcionalidades

- Gestión de Empleados
- Gestión de Departamentos  
- Gestión de Proyectos
- Operaciones CRUD completas

## Tecnologías

- Python 3.x
- Base de datos SQL
- Arquitectura en capas
- POO y Herencia

## Instalación

```bash
pip install -r requirements.txt
python main.py
```
